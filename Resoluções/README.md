# 📂 Resoluções

Esta pasta contém todas as minhas resoluções de problemas do LeetCode.

Cada problema possui sua própria subpasta nomeada com o número e o título do problema, por exemplo:

```
0001-two-sum/
0002-add-two-numbers/
...
```

Dentro de cada subpasta você encontrará as soluções implementadas nas linguagens disponíveis, acompanhadas de comentários e notas sobre o raciocínio utilizado.
