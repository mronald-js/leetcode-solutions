# 🧩 Leetcode Solutions

Bem-vindo ao meu repositório de resoluções de problemas do [LeetCode](https://leetcode.com/)!

## 📌 Sobre este repositório

Este repositório tem como objetivo registrar minhas resoluções de problemas do LeetCode, servindo como um diário de aprendizado e evolução na lógica de programação.

Para cada problema, busco não apenas apresentar o código final, mas também compartilhar o **raciocínio por trás da solução**: os pensamentos iniciais, as abordagens tentadas, os obstáculos encontrados e as conclusões tiradas. Acredito que documentar o processo de pensar é tão valioso quanto a solução em si.

## 🎯 Objetivo

- Exercitar e aprimorar a **lógica de programação**
- Desenvolver fluência em múltiplas linguagens de programação
- Registrar o processo de raciocínio (monólogos e reflexões) para cada problema
- Criar um portfólio de soluções comentadas e bem explicadas

## 💻 Linguagens utilizadas

As soluções serão implementadas nas seguintes linguagens, **nesta ordem de prioridade**:

1. 🐍 **Python 3** — linguagem principal, pela sua legibilidade e facilidade de prototipação
2. 🟦 **TypeScript** — para reforçar conceitos de tipagem e desenvolvimento web moderno
3. 🐹 **Golang** — para explorar performance e concorrência
4. ☕ **Java** — para exercitar orientação a objetos e padrões clássicos

## 🧠 Metodologia

Para cada problema, tento seguir este fluxo:

1. **Entendimento** — Leitura cuidadosa do enunciado e identificação das restrições
2. **Exploração** — Pensar em exemplos, casos extremos e primeiras ideias (força bruta incluída)
3. **Refinamento** — Otimizar a solução pensando em complexidade de tempo e espaço
4. **Implementação** — Escrever o código de forma clara e comentada
5. **Reflexão** — Registrar o que aprendi e o que poderia ter feito diferente
